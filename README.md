# zimg

![](docs/upload.gif)

zimg 图床cli工具:

1. 目前支持「github」;
2. 交互性设置图床源;

## 使用

全部命令

```bash
Usage of zimg:
 zimg <file>    上传图片
 zimg test      检查配置文件, 没有配置文件则创建一份配置文件[~/.zimgrc]
 zimg current   查看当前使用的图床源
 zimg use       选择使用的图床源
 zimg get       选择查看某个图床源配置
 zimg set       选择设置某个图床源配置
 zimg help      查看帮助
```
